# 生物信息学 Python 教程

> 面向零基础生物学本科生，从 `print("Hello")` 到机器学习分类乳腺癌。

## 这是什么？

一套 **10 章** 的 Python 入门教程，专为 **生物学/生命科学专业** 学生设计。

所有概念都用生物学类比来解释——变量是贴了标签的试管，函数是催化反应的酶，循环是 PCR 扩增。你不需要任何编程基础，只需要对生物学有基本了解。

## 如何使用

**第一步：下载教程**

从老师那里拿到教程文件夹（`bio-python`），放在你的桌面上就行。

**第二步：用 VSCode 打开**

1. 打开 VSCode
2. 点击菜单栏 → **文件** → **打开文件夹**
3. 选择桌面上的 `bio-python` 文件夹，点"确定"

> 如果你还没安装 uv 和 VSCode，别急——第 1 章的讲义里有详细的手把手安装指南，跟着做就行。

**第三步：安装依赖**

在 VSCode 底部终端（按 `` Ctrl+` `` 打开）输入以下命令，按 Enter 运行：

```bash
uv sync
```

这一步会自动安装教程所需的 Python 和所有工具包。只需要运行一次，以后就不用再装了。

**第四步：开始学习（每章重复这个流程）**

1. 📖 打开 `lecture.md` 讲义，阅读概念讲解
2. ▶️ 在 VSCode 底部终端输入 `uv run python chXX/demo.py`，观察代码运行效果
3. ✏️ 打开 `exercises.py`，动手尝试完成练习
4. 🔍 遇到问题？回头看讲义对应部分，或者在代码里加 `print()` 调试

> `solutions.py` 是教师版参考答案，建议先独立完成 `exercises.py`，再对照答案检查。

## 运行示例

```bash
# 运行第 1 章演示
uv run python ch01_hello_python/demo.py

# 运行第 1 章教师版参考答案
uv run python ch01_hello_python/solutions.py

# 运行第 10 章毕业项目
uv run python ch10_bio_ml_project/demo.py
```

## 关于运行产物

运行 `demo.py` 或 `solutions.py` 时，部分章节会生成 PNG、CSV、FASTA 或文本报告文件。
这些文件只是本地运行产物，不需要提交到仓库，`.gitignore` 已经默认忽略它们。
